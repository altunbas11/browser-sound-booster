Browser Sound Booster does NOT collect, store, or share any personal information.

All volume settings are stored locally on your device only.

Data Collection

NO DATA IS COLLECTED

  • No personal information
  • No browsing history
  • No location data
  • No cookies or tracking
  • No external server communication

Data Storage

The extension only stores:

  • Volume levels (100%, 200%, etc.)
  • Mute status
  • Settings per website domain

All data is stored locally using Chrome's storage API and never transmitted anywhere.

Third-Party Services

This extension does not use any third-party services, analytics, or external APIs.

Permissions

The extension requests minimal permissions only necessary for audio boosting:

  • activeTab - Access current tab for audio processing
  • storage - Save volume settings locally
  • tabs - Communicate between extension popup and content script
  • <all_urls> - Work on any website with media content

Contact

If you have questions about this privacy policy, please check the extension's GitHub repository.

Changes to This Policy

This privacy policy may be updated. Any changes will be reflected in the extension's repository.

This extension follows Chrome Web Store Developer Program Policies and collects no user data.
